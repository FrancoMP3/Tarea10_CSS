Margins Example

margin-top

En CSS, la propiedad margin-top establece el espacio entre el borde superior de un elemento y los elementos adyacentes.

Este espacio de margen separa el elemento de otros elementos ubicados arriba.

margin-right

En CSS, la propiedad margin-right establece el espacio entre el borde derecho de un elemento y los elementos adyacentes.

Este espacio de margen separa el elemento de otros elementos ubicados a la derecha.

margin-bottom

En CSS, la propiedad margin-bottom establece el espacio entre el borde inferior de un elemento y los elementos adyacentes.

Este espacio de margen separa el elemento de otros elementos ubicados abajo.

margin-left

En CSS, la propiedad margin-left establece el espacio entre el borde izquierdo de un elemento y los elementos adyacentes.

Este espacio de margen separa el elemento de otros elementos ubicados a la izquierda.
