Visible

La propiedad `overflow: visible` permite que el contenido desborde del elemento, es decir, el contenido se muestra aunque esté fuera del área del elemento.

Hidden

La propiedad `overflow: hidden` oculta el contenido que desborda del elemento, es decir, el contenido que no cabe dentro del área del elemento no se mostrará.

Scroll

La propiedad `overflow: scroll` muestra barras de desplazamiento tanto horizontal como verticalmente, permitiendo desplazar el contenido que no cabe dentro del área del elemento.

Auto

La propiedad `overflow: auto` muestra barras de desplazamiento solo cuando sea necesario, es decir, solo cuando el contenido desborda del área del elemento o cuando es el caso de que el texto es demasiado grande como para que solo se vea entonces ahi es donde aparece la barra para poder deslizar y ver el resto del contenido.
