Layout

La propiedad `display: inline-block` se utiliza para cambiar el diseño de un elemento de bloque a un elemento en línea que todavía conserva las propiedades de un bloque. Esto significa que el elemento se comportará como un elemento en línea, permitiendo que otros elementos se coloquen en la misma línea, pero también mantendrá las propiedades de tamaño y espaciado de un bloque.
