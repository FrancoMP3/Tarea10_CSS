EJERCICIO 1

La funcion RGB se utiliza para dar o designar un color a un texto parrafo o titulo mediante el uso de tres colores que son: rojo, verde y azul, cambiando el color dando numeros a cada uno en un rango de 0 a 255.

En CCS (Custom Computer Services), la función RGB se utiliza para definir un color específico en un sistema de pantalla o visualización. La función RGB toma tres parámetros: el valor de intensidad de rojo (R), el valor de intensidad de verde (G) y el valor de intensidad de azul (B). Estos valores van desde 0 hasta 255, donde 0 representa la ausencia total del color y 255 representa la máxima intensidad del color.

Al combinar diferentes valores de intensidad de rojo, verde y azul, se pueden crear una amplia gama de colores. Por ejemplo, si se establecen los valores R=255, G=0 y B=0, se obtiene el color rojo puro, mientras que si se establecen R=0, G=255 y B=0, se obtiene el color verde puro.


En CSS, los colores se pueden definir utilizando valores hexadecimales, que consisten en una combinación de números y letras que representan la cantidad de rojo, verde y azul en un color específico. Por ejemplo, #FF0000 representa el color rojo puro, donde FF representa el valor máximo de intensidad para el color rojo.

La representación hexadecimal de colores en CSS proporciona una forma precisa y versátil de especificar colores para los elementos de una página web. Esto permite a los desarrolladores y diseñadores tener un control detallado sobre la apariencia visual de sus sitios, desde el color de fondo de una sección hasta el tono de un texto específico. Además, al utilizar códigos hexadecimales, se puede garantizar una consistencia en el color en diferentes dispositivos y navegadores web.

Los códigos hexadecimales se aplican en diversos aspectos del diseño web, como la definición del color de fondo de una página, la tonalidad de los textos, el color de los enlaces, los bordes de los elementos y mucho más. Además, se pueden utilizar para crear efectos visuales complejos mediante la manipulación de gradientes de color o la transparencia. En resumen, la aplicación de los colores hexadecimales en CSS permite una personalización detallada y precisa del diseño de una página web, contribuyendo a una experiencia visual atractiva y coherente para los usuarios.
