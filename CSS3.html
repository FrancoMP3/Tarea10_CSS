Ejercicio 3

Esta propiedad CSS controla el radio de los bordes de un elemento. Se utiliza para redondear los bordes de un elemento HTML, creando esquinas suavizadas en lugar de esquinas afiladas. Se puede aplicar a los cuatro bordes individuales o a todos a la vez, y se especifica utilizando valores en píxeles, porcentajes o valores relativos.

La propiedad border en CSS define el borde de un elemento HTML. Se utiliza para establecer el ancho, el estilo y el color del borde alrededor de un elemento. Esta propiedad se puede abreviar especificando los tres componentes del borde (ancho, estilo y color) en un solo valor, o se puede dividir en border-width, border-style y border-color para un control más detallado.

El valor dotted de la propiedad border-style en CSS crea un borde con puntos. Cada punto es redondo y está separado del siguiente por la misma distancia. Este estilo de borde es útil para crear efectos decorativos sutiles o para resaltar ciertos elementos en el diseño.

El valor dashed de la propiedad border-style en CSS crea un borde con líneas discontinuas. Cada línea es uniformemente espaciada y tiene la misma longitud. Este estilo de borde es comúnmente utilizado para crear bordes decorativos o para separar secciones en el diseño web.

El valor solid de la propiedad border-style en CSS crea un borde sólido y continuo alrededor de un elemento HTML. Este es el estilo de borde más básico y se utiliza comúnmente para definir bordes simples y claros en los elementos de una página web.

El valor double de la propiedad border-style en CSS crea un borde con dos líneas sólidas separadas por un espacio. Este estilo de borde es más grueso que un borde sólido normal y se utiliza para agregar énfasis o acentuar los elementos en el diseño.

El valor groove de la propiedad border-style en CSS crea un borde tridimensional que parece estar tallado en el elemento. Tiene un aspecto de relieve con un borde más oscuro en la parte superior e inferior y un borde más claro en los lados. Este estilo de borde se utiliza a menudo para crear efectos de profundidad y dimensiones en el diseño.

El valor ridge de la propiedad border-style en CSS es similar a groove, pero al revés. Crea un borde tridimensional con un aspecto de relieve donde los bordes superiores e inferiores son más claros y los lados son más oscuros. Se utiliza para resaltar los elementos y crear efectos de relieve en el diseño.

El valor inset de la propiedad border-style en CSS crea un borde con un efecto de sombra interna, que da la apariencia de que el contenido del elemento está incrustado en el contenedor. Este estilo de borde se utiliza para crear botones o elementos con apariencia tridimensional en el diseño.

El valor outset de la propiedad border-style en CSS es el opuesto de inset. Crea un borde con un efecto de sombra externa, que da la apariencia de que el contenido del elemento sobresale del contenedor. Se utiliza para crear elementos con una apariencia elevada o resaltada en el diseño.

El valor none de la propiedad border-style en CSS elimina completamente el borde alrededor de un elemento HTML. Se utiliza cuando no se desea ningún borde alrededor del elemento.

El valor hidden de la propiedad overflow en CSS oculta el contenido que se desborda del contenedor. Se utiliza para ocultar cualquier contenido que exceda los límites del elemento, en lugar de mostrar barras de desplazamiento. En el contexto de los bordes, este valor no se aplica directamente, pero puede utilizarse para ocultar los desbordamientos de los elementos si así se desea.
