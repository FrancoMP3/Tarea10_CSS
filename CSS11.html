Borders

La propiedad CSS `border` se utiliza para establecer el ancho, el estilo y el color del borde de un elemento.

En el contexto de una tabla, la propiedad `border` se aplica a las celdas (`th`, `td`) para definir los bordes de las celdas.

Size

La propiedad CSS `width` y `height` se utilizan para establecer el ancho y la altura de un elemento, respectivamente.

En el contexto de una tabla, estas propiedades se pueden aplicar a la tabla en sí misma o a las celdas para controlar su tamaño.

Alignment

La propiedad CSS `text-align` se utiliza para alinear el contenido de un elemento en relación con el borde del elemento.

En el contexto de una tabla, esta propiedad se puede aplicar a las celdas (`th`, `td`) para alinear el texto dentro de ellas.
