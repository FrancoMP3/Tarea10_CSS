Color

La propiedad CSS `color` define el color del texto en un elemento HTML.

Se puede especificar utilizando un nombre de color, un valor hexadecimal o un valor RGB.

Text-Align

La propiedad CSS `text-align` establece la alineación del texto en un elemento HTML.

Se puede usar para alinear el texto a la izquierda, derecha, centrado o justificado.

Text-Decoration

La propiedad CSS `text-decoration` añade decoración al texto en un elemento HTML.

Se puede usar para subrayar, tachar, o hacer que el texto sea subíndice o superíndice.

Text-Transform

La propiedad CSS `text-transform` especifica cómo se transforma el texto en un elemento HTML.

Se puede usar para convertir el texto a mayúsculas, minúsculas o a capitalizar cada palabra.

Letter-Spacing

La propiedad CSS `letter-spacing` controla el espacio entre caracteres en un elemento HTML.

Se puede usar para ajustar el espaciado entre letras para mejorar la legibilidad o el diseño.
